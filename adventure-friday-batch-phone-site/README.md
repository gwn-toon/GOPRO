# Adventure Friday Batch Phone Site

This folder is ready to upload to GitHub Pages.

## Files

- `index.html`: the standalone phone helper page
- `.nojekyll`: tells GitHub Pages to serve the static file directly

## Fastest GitHub Pages setup

1. Create a new GitHub repository.
2. Upload the contents of this folder to the repository root.
3. Open the repository on GitHub.
4. Go to `Settings` -> `Pages`.
5. Under `Build and deployment`:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
6. Save.
7. Wait a minute.
8. Your page will appear at:

`https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPO-NAME/`

## Example

If your GitHub username is `toonvandevyver` and your repo is called
`adventure-friday-batch-phone`, the URL becomes:

`https://toonvandevyver.github.io/adventure-friday-batch-phone/`

## Phone use

1. Open the URL in Safari on your phone.
2. Tap the share button.
3. Choose `Add to Home Screen`.
4. Open it from your home screen like an app.

## Update later

If you replace `index.html` in the repo with a newer version, GitHub Pages will
refresh the site automatically after a short delay.
